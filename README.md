
# Introduction
해당 문서들은 자대의 인공지능 연구실에서 경험했던 장기프로젝트에서 코드리뷰를 하기에 적합한 일부를 발췌한 것입니다.  
프로젝트는 크게 다음과 같은 순서로 진행 되었습니다.  
 **크롤링 -> 정제 -> 텍스트 마이닝 -> 추천 알고리즘 구현 -> mirror site에 접목**  
Repository에 push 되어있는 디렉토리들은, 그 중 크롤링, 텍스트마이닝, 웹과 관련된 코드들을 정리해 놓은 것입니다.
